# Pomodoro Timer

A simple, focused timer to help you manage work and breaks using the Pomodoro Technique.

## Features
*   **Work/Break Cycles:** Easily switch between 25-minute work sessions and 5-minute breaks.
*   **Custom Session Titles:** Set a custom title for each session (e.g., "Deep Work", "Study").
*   **Session History:** Track and view your completed work sessions.
*   **Data Export:** Export your session history as a CSV file.

## How to Run
1.  Open the `index.html` file in your web browser.
2.  Start the timer and manage your work/break cycles.

## Technology
This application is built using HTML, CSS, and JavaScript.
